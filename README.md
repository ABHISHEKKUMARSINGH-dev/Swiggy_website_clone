# CraveWave 🍔

A front-end web application replicating the core user interface of the Swiggy landing page. **CraveWave** focuses on delivering a clean, visually appealing, and intuitive UI that matches the signature look and feel of the popular food delivery platform.

## 🌍 Live Demo

Check out the live deployed site here: **[CraveWave on Vercel](https://cravewave-brown.vercel.app/)**

## 🚀 Overview

This repository contains the HTML and CSS necessary to render the main hero section of our food delivery platform, complete with the branding, location selection, search functionality, and category navigation (Food, Instamart, Dineout). 

## 🛠️ Tech Stack

* **HTML5:** Semantic structuring of the landing page elements.
* **CSS3:** Custom styling, flexbox layouts, hover animations, and the signature `#fc8019` color scheme.
* **Deployment:** Hosted live on Vercel for fast, reliable global access.

## ✨ Features

* **Authentic UI:** Accurately mimics modern food delivery typography, button styles, and layout.
* **Interactive Elements:** Smooth hover scaling effects on the primary category banners to enhance user experience.
* **Structured Layout:** Utilizes CSS Flexbox and absolute positioning to align navigation links, calls-to-action, and decorative food imagery precisely.

## 💻 How to Run Locally

If you want to run or edit this project on your own machine:

1.  Clone this repository to your local Windows environment:
    ```bash
    git clone [https://github.com/yourusername/crave_wave.git](https://github.com/yourusername/crave_wave.git)
    ```
2.  Navigate to the project directory.
3.  Double-click the `INDEX.HTML` file, or right-click and select **Open with** to view it in Chrome, Edge, or your preferred browser.

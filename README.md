# Swiggy Landing Page Clone 🍔

A front-end web application replicating the core user interface of the Swiggy landing page. This project focuses on delivering a clean, visually appealing, and intuitive UI that matches the signature look and feel of the popular food delivery platform.

## 🚀 Overview

This repository contains the HTML and CSS necessary to render the main hero section of Swiggy, complete with the branding, location selection, search functionality, and category navigation (Food, Instamart, Dineout). 

## 🛠️ Tech Stack

* **HTML5:** Semantic structuring of the landing page elements.
* **CSS3:** Custom styling, flexbox layouts, hover animations, and Swiggy's signature `#fc8019` color scheme.

## ✨ Features

* **Authentic UI:** Accurately mimics Swiggy’s typography, button styles, and layout.
* **Interactive Elements:** Smooth hover scaling effects on the primary category banners to enhance user experience.
* **Structured Layout:** Utilizes CSS Flexbox and absolute positioning to align navigation links, calls-to-action, and decorative food imagery precisely.

## 💻 How to Run Locally

Since this is a static front-end project, no complex server setup is required.

1.  Clone this repository to your local Windows machine:
    ```bash
    git clone [https://github.com/yourusername/swiggy-clone.git](https://github.com/yourusername/swiggy-clone.git)
    ```
2.  Navigate to the project directory.
3.  Simply double-click the `INDEX.HTML` file, or right-click and select **Open with** to view it in Chrome, Edge, or your preferred browser.
